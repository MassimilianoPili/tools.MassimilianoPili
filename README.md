# Dev Tools

Browser-based developer utilities. No frameworks, no build step -- just open the HTML files.

## Tools

| Tool | Description | Live |
|------|-------------|------|
| [Mongo Converter](mongo/) | MongoDB &harr; Spring `@Aggregation` bidirectional converter | [Open](https://tools.massimilianopili.com/mongo) |
| [Base64](base64/) | Base64 encoder/decoder with file type detection | [Open](https://tools.massimilianopili.com/base64) |
| [Prettifier](prettifier/) | JSON & cURL prettifier with syntax highlighting | [Open](https://tools.massimilianopili.com/prettifier) |
| [JWT](jwt/) | JWT decoder & editor | [Open](https://tools.massimilianopili.com/jwt) |

## Run locally

```bash
git clone https://github.com/MassimilianoPili/tools.MassimilianoPili.git
cd tools.MassimilianoPili
python -m http.server 8000
# open http://localhost:8000/mongo/
```

## Tech Stack

Pure HTML5, CSS3, and vanilla JavaScript. No dependencies, no build system.
